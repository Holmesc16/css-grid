Grid uses tracks (collectively the columns+rows) which, within a container, allow for a template-level stying for said columns/rows.

We can also use grid-gap for creating marginal space between tracks

when a fixed pixel (or other value like rem, pct, etc.) is assigned, it will remain fixed to that value in responsive view. When auto is assigned within these same values, it will adjust according to the remaining space alloted within that track. We can use functions like grid-template-columns: repeat(6, 100px), which will create six columns all of 100px in width.
